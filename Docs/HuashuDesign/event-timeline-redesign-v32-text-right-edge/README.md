- 单独新建 v32 目录，没有覆盖 v31 或其他版本。

- 以推荐页卡片正文右边界为基准。

- 不使用渐变色。
- 不使用不规则多边形色块。

- 保留当前截图里的入口、页面结构、分类文案和业务规则。
