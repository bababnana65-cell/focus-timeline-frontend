# 事件时间轴 App · iOS Journal Date Column Topics v24 视觉规范
> 日期：2026-04-28  
> 范围：基于 v23 的专题卡片局部调整，不修改 Flutter 代码  
> 关键调整：我的关注卡片日期进入左侧图标列，并与最新动态摘要同一行

## 本版调整

v24 只调整“我的关注”专题卡片：

- 左侧列包含专题图标和日期。
- 日期位于图标下方。
- 右侧正文列保留标题、专题摘要、最新动态摘要。
- 日期与最新动态摘要在视觉上处于同一行。
- 未读卡片的日期和最新动态摘要统一使用珊瑚提醒色。
- 普通卡片的日期和最新动态摘要统一使用蓝色。
- 右侧箭头、状态标签、红点规则不变。

## 组件规则

- `topic-leading`：左侧列，垂直摆放图标和日期。
- `topic-date`：11.5px，居中，不换行。
- `topic-latest-row`：12px，和日期同一阅读行。
- 未读态：`topic-date`、`topic-latest-row`、`latest` 使用同一提醒色。
- 准备中态：日期使用准备中语义色。

## 继承规则

- 时间轴顶部统计区保持单行胶囊。
- 时间轴节点圆点保持 v23 放大样式。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
