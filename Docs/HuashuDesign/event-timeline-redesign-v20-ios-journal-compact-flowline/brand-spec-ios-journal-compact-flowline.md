# 事件时间轴 App · iOS Journal Compact Flowline v20 视觉规范
> 日期：2026-04-28  
> 范围：基于 v19 的时间轴局部调整，不修改 Flutter 代码  
> 关键调整：压缩左侧日期列和节点流线的宽度，为右侧节点信息释放阅读空间

## 本版调整

v20 继承 v19 的节点流线结构，但减少左侧横向占位：

- 时间轴左侧总占位从 82px 压缩到约 70px。
- 日期列宽度从 48px 压缩到 43px，仍右对齐显示在节点前。
- 节点流线轨道从 24px 收窄到 18px。
- 节点数量圆点从 22px 收窄到 20px。
- 连接短线缩短，减少对卡片起始位置的占用。
- 右侧事件卡片获得更多正文宽度，标题换行更少。

## 时间轴组件规则

- 日期文字：统一天蓝色，右对齐，保持在流线左侧。
- 流线：统一青柠绿，贯穿节点组，不使用渐变。
- 流线底色：极浅青柠色圆角轨道，但尽量窄。
- 节点摘要：统一 `--muted`，不因节点重要程度变色。
- 重大节点数量点：统一珊瑚色。
- 普通节点数量点：统一时间轴主题青柠色。
- 卡片标题：保持正文主色，优先阅读。

## 继承规则

- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 新建流程：`新建时间线 -> 关键词 -> AI 扩写 -> 专题定义确认`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
