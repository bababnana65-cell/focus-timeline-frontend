# 事件时间轴 App 视觉重设计 · iOS Journal Compact Flowline v20

本目录是 `huashu-design` 交付物，只做视觉设计，不修改 Flutter 业务代码。

## 本版方向

- 单独新建 v20 目录，没有覆盖 v19 或其他版本。
- 以 v19 为基础，只压缩时间轴左侧日期列和节点流线宽度。
- 时间轴仍保持“前置日期 + 节点流线 + 事件卡片”。
- 左侧占位从 82px 收窄到约 70px，让右侧节点卡片正文更宽，同时避免日期折行失控。
- 日期仍显示在节点前方，节点数量圆点仍挂在流线上。
- 保留 v17/v19 的整体色彩、语义色规则和页面视觉基调。
- 不使用渐变色。
- 不使用不规则多边形色块。
- 不使用黑色或深色填充块；深色只用于正文文字和必要线性图标。
- 保留当前截图里的入口、页面结构、按钮文案、分类文案和业务规则。

## 文件

- `event-timeline-ios-journal-compact-flowline.html`：高保真移动端设计板。
- `brand-spec-ios-journal-compact-flowline.md`：紧凑节点流线视觉规范。
- `screenshots/design-board.png`：完整设计板 PNG。
- `screens/*.png`：单页高保真 PNG。
- `figma-handoff.json`：Figma 重建参考 JSON，不是原生 `.fig` 文件。

- 我的关注：普通列表 / 有新动态 / 当前专题 / 空列表
