# 事件时间轴 App · V33 Timeline Tab v59 视觉规范
> 日期：2026-05-02  
> 范围：基于 v58，只调整底部中间时间轴 Tab 图标与文字，不修改 Flutter 代码  
> 关键调整：采用 v33 的 `clock` 图标方案，并显示 `时间轴` 文字

## 本版调整

v59 只调整底部导航中间 `时间轴` Tab：

- 第二个底部 Tab 使用 v33 的 `clock` symbol：`<use href="#clock" />`。
- 显示文字 `时间轴`，不再使用纯图标入口。
- `.bottom .nav:nth-child(2)` 增加 `timeline-v33-tab` 类，仅用于稳定图标尺寸和槽位。
- `.wrap` 尺寸恢复为 `54px × 34px`，和普通底部 Tab 的视觉节奏一致。
- 图标尺寸为 `20px × 20px`。
- 选中态仍由全局 `.nav.active .wrap` 和当前页面主题变量控制。
- 未选中态仍使用 `var(--muted)`，和 `推荐` 未选中图标保持同类灰色。

## 不变项

- 不改变底部三 Tab 数量和语义：`我的关注 / 时间轴 / 推荐`。
- 不改变推荐页、我的关注页、时间轴页、新建流程、登录弹层和额度提示。
- 不改变时间轴节点卡片、轨道、初始节点、最新节点虚线和准备中状态。
- 不改变接口、状态流、业务规则或 Flutter 实现。

## 继承说明

本版继承 v58 的全部页面结构和 v57 以前的累计视觉调整，只替换底部中间 Tab 的图标呈现方式。v33 的参考点是：

- 使用内置 `clock` 图标，而不是自定义 App 图标。
- 图标下方保留 `时间轴` 文字。
- 选中态通过浅色圆角背景和主题色图标表达。

## 开发映射

Flutter 实现时只需要把底部中间 Tab 的视觉映射为：

- icon: `clock` / 表盘样式
- label: `时间轴`
- selected color: 当前时间轴主题绿色
- unselected color: 当前底部导航灰色
- selected background: 当前 Tab 选中浅色胶囊

不需要新增路由、状态、接口、通知或入口。
