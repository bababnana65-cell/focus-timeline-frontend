# 事件时间轴 App 视觉重设计 · Current No Label v31

本目录是 `huashu-design` 交付物，只做视觉设计，不修改 Flutter 业务代码。

## 本版方向

- 单独新建 v31 目录，没有覆盖 v30 或其他版本。
- 以 v30 为基础，只调整“我的关注 · 当前专题”卡片。
- 去掉当前专题卡片右侧的“正在查看”文字标签。
- 当前专题状态改为只通过卡片底色、边框和阴影表达。
- 去掉该标签占用的右侧列，让最新节点消息获得更多阅读宽度。
- 保留 v30 的推荐页调整：卡片内类型标签隐藏，“已关注”在左侧图标下方。
- 保留未读红点提示。
- 推荐页左滑关注/取消关注逻辑继承 v25。
- 不使用渐变色。
- 不使用不规则多边形色块。
- 不使用黑色或深色填充块。
- 保留当前截图里的入口、页面结构、分类文案和业务规则。
