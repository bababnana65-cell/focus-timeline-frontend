# 事件时间轴 App · Current No Label v31 视觉规范
> 日期：2026-04-29  
> 范围：基于 v30 的“我的关注”当前专题卡片局部调整，不修改 Flutter 代码  
> 关键调整：去掉“正在查看”文字标签，保留卡片态表达当前专题

## 本版调整

v31 只调整当前专题卡片：

- 当前专题卡片不再显示“正在查看”标签。
- 当前专题通过淡色背景、边框和轻阴影表达。
- 当前专题卡片不保留右侧标签列，正文区域更宽。
- 组件规范页不再展示“正在查看”状态胶囊。
- 继承 v30：推荐卡片左侧图标保留，已关注在图标下方。

## 组件规则

- `topic.current`：双列布局，左图标列 + 正文列，不显示右侧状态列。
- `topic.current .side`：隐藏。
- 当前专题识别：使用背景、边框、阴影，不使用文字标签。
- `topic-latest-row`：继续作为主阅读信息，获得更宽行长。

## 继承规则

- 推荐卡片继承 v30：左侧原图标保留，已关注放到图标下方。
- 时间轴顶部统计区保持单行胶囊。
- 时间轴节点圆点保持 v23 放大样式。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
