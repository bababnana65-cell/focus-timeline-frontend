# 事件时间轴 App · iOS Journal Larger Nodes v23 视觉规范
> 日期：2026-04-28  
> 范围：基于 v22 的时间轴局部调整，不修改 Flutter 代码  
> 关键调整：放大左侧流线上的节点圆点

## 本版调整

v23 只调整时间轴节点圆点：

- 节点圆点从 20px 放大到 24px。
- 数字字号从默认小号提升到 12px。
- 圆点 left/top 重新校准，保证圆心仍然落在流线上。
- 普通节点仍使用时间轴主题色。
- 重大节点仍使用珊瑚色。
- 日期标签、单行统计条、节点卡片布局保持 v22 不变。

## 组件规则

- 节点圆点：24px，居中挂在线上。
- 普通节点：白底、青柠文字、浅青柠描边。
- 重大节点：浅珊瑚底、珊瑚文字、浅珊瑚描边。
- 流线：保持细线，不随圆点一起加粗，避免左侧过重。

## 继承规则

- 时间轴顶部统计区保持单行胶囊。
- 日期标签位于节点卡片顶部。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
