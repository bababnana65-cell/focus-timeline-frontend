# 事件时间轴 App · iOS Journal Topic Alignment v26 视觉规范
> 日期：2026-04-29  
> 范围：基于 v25 的“我的关注”卡片局部调整，不修改 Flutter 代码  
> 关键调整：日期与最新动态摘要对齐，移除重复提示和右箭头

## 本版调整

v26 只调整“我的关注”专题卡片：

- 日期保留在左侧图标列下方，但进一步下移。
- 日期与右侧最新动态摘要在同一行对齐。
- 移除右侧 `新动态` 文字。
- 未读状态只保留红点和摘要高亮。
- 移除右侧 `>` 箭头。
- 右侧状态/箭头区域收窄，让正文宽度接近推荐页卡片。

## 组件规则

- `topic-date`：11.5px，居中，不换行，与摘要行对齐。
- `topic-latest-row`：12px，和日期同一阅读行。
- 未读态：日期与摘要统一珊瑚色。
- 普通态：日期与摘要统一蓝色。
- 未读提示：只使用红点，不显示“新动态”文字。
- 进入详情：整张卡片点击，不依赖右侧箭头。

## 继承规则

- 推荐卡片继承 v25：无常驻关注/查看按钮，左滑露出关注/取消关注。
- 时间轴顶部统计区保持单行胶囊。
- 时间轴节点圆点保持 v23 放大样式。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
