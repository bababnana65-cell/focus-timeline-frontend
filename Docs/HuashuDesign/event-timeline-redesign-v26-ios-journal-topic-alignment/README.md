# 事件时间轴 App 视觉重设计 · iOS Journal Topic Alignment v26

本目录是 `huashu-design` 交付物，只做视觉设计，不修改 Flutter 业务代码。

## 本版方向

- 单独新建 v26 目录，没有覆盖 v25 或其他版本。
- 以 v25 为基础，只调整“我的关注”专题卡片。
- 日期下移，与右侧最新动态摘要同一行对齐。
- 移除未读卡片右侧的“新动态”文字，避免和红点重复。
- 移除专题卡片右侧 `>` 箭头。
- 收窄右侧状态/箭头区域，把更多宽度还给标题、摘要和最新动态。
- 保留未读红点提示。
- 推荐页左滑关注/取消关注逻辑继承 v25。
- 不使用渐变色。
- 不使用不规则多边形色块。
- 不使用黑色或深色填充块。
- 保留当前截图里的入口、页面结构、分类文案和业务规则。
