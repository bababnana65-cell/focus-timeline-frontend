# 事件时间轴 App · iOS Journal Swipe Recommend v25 视觉规范
> 日期：2026-04-29  
> 范围：基于 v24 的局部调整，不修改 Flutter 代码  
> 关键调整：推荐卡片取消常驻按钮，改为左滑动作；我的关注日期下移对齐摘要

## 我的关注卡片

- 日期仍在左侧图标列下方。
- 日期下移，与右侧最新动态摘要处于同一行。
- 未读卡片里日期和摘要统一珊瑚提醒色。
- 普通卡片里日期和摘要统一蓝色。
- 右侧箭头、状态标签、红点规则不变。

## 推荐卡片

- 取消常驻 `关注`、`已关注`、`查看` 按钮。
- 整张推荐卡片作为查看入口。
- 左滑卡片时露出操作：
  - 未关注：`关注`
  - 已关注：`取消关注`
- 已关注专题在推荐类型标签左侧显示 `已关注`。
- 推荐类型标签保留当前页面分类文案，例如 `当前热门`、`可能关心`、`随机看看`、`历史记录`。
- 关注人数仍保留在卡片正文里。

## 左滑状态

- 默认态：只显示专题图标、标签、标题、摘要、关注人数。
- 左滑态：右侧露出 78px 操作区。
- 已关注卡片左滑动作使用珊瑚色 `取消关注`。
- 未关注卡片左滑动作使用页面主题色 `关注`。

## 继承规则

- 时间轴顶部统计区保持单行胶囊。
- 时间轴节点圆点保持 v23 放大样式。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
