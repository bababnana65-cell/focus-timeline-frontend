# 事件时间轴 App 视觉重设计 · No Polygon Blocks v7

本目录是 `huashu-design` 交付物，只做视觉设计，不修改 Flutter 业务代码。

## 本版变化

- 基于上一版单独新建 v7 目录，没有覆盖旧文件。
- 取消不规则多边形色块和装饰性色块。
- 不使用渐变色。
- 不使用黑色或深色填充块；深色只用于正文文字和必要图标线条。
- 重点状态改由留白、卡片层级、红点、状态标签、细边框和浅色底表达。
- 保留当前截图里的入口、页面结构、按钮文案、分类文案和业务规则。

## 文件

- `event-timeline-no-polygon-blocks.html`：高保真移动端设计板。
- `brand-spec-no-polygon-blocks.md`：无多边形色块版视觉规范。
- `screenshots/design-board.png`：完整设计板 PNG。
