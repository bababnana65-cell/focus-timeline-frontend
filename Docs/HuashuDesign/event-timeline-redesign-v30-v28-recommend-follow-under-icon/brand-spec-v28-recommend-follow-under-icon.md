# 事件时间轴 App · Recommend Follow Under Icon v30 视觉规范
> 日期：2026-04-29  
> 范围：基于 v28 的推荐卡片局部调整，不修改 Flutter 代码  
> 关键调整：去掉卡片内推荐类型标签，把已关注状态移到左侧图标下面

## 本版调整

v30 只调整推荐卡片状态摆放：

- 保留推荐卡片左侧现有图标。
- 不再在卡片正文上方显示推荐类型小标签。
- 已关注状态放到左侧图标下面。
- 未关注卡片左侧只显示原图标。
- 标题、摘要、关注人数的字体、颜色和排版保持 v28 不变。
- 继承 v28：我的关注卡片继续突出最新节点消息。

## 组件规则

- `rec-leading`：左列容器，垂直放置原图标和已关注状态。
- `rank`：保留 v28 的原图标视觉。
- `followed-tag`：保留 v28 的绿色胶囊样式，放在图标下方。
- `rec-tag`：不显示，推荐分类由顶部 Tab 表达。
- `rec` 正文：标题、摘要、关注人数格式不变。

## 继承规则

- 推荐卡片继承 v25/v28：无常驻关注/查看按钮，左滑露出关注/取消关注。
- 时间轴顶部统计区保持单行胶囊。
- 时间轴节点圆点保持 v23 放大样式。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
