# 事件时间轴 App · 视觉重设计规范
> 版本：logic locked v3  
> 日期：2026-04-28  
> 输入：用户提供的 7 张当前 App 截图与本次任务书  
> 范围：视觉设计交付，不修改 Flutter 代码

## 设计原则

- 只重做视觉，不改页面逻辑、入口位置、文案、业务规则和接口状态流。
- 推荐页分类固定为：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部导航固定为：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单固定为：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、设置、编辑、删除、归档等入口。

## 视觉方向

方向名：**Airy Timeline / 清爽轻彩时间轴**

- 背景：浅蓝白与微薄荷灰，避免后台系统感和沉重新闻感。
- 状态色：浅蓝表示主行动与当前状态，薄荷表示准备中与成功，淡紫表示轻提示与排名，珊瑚表示新动态。
- 卡片：低半径、轻阴影、少边框，保留明确点击区域。
- 时间线：日期、节点数量圆点、节点卡片三段式，保留当前阅读结构。

## 字体层级

- 页面标题：21px / 850 / line-height 1.2
- 卡片标题：16px / 850 / line-height 1.32
- 正文摘要：13.5px / 560 / line-height 1.48
- 元信息：12px / 850
- 底部 Tab：11px / 850

## 组件规则

- 顶部栏：左右按钮 44px，标题居中，长标题单行省略。
- 底部 Tab：高度 76px，三入口固定；红点只出现在“我的关注”。
- 推荐分类切换：4 等分胶囊，文案固定。
- 推荐卡片：左侧排名或闪光图标，中间标题/摘要/关注人数，右侧关注状态和查看。
- 关注卡片：左侧专题图标，中间标题/摘要/日期/最新动态，右侧箭头。
- 时间线节点：左侧日期和数量圆点，右侧节点卡片，展开后在卡片内部显示动态摘要。
- 底部 Sheet：用于新建时间线，顶部 grabber，主按钮 48px。
- Toast：底部 48px 高，失败时保留旧内容。

## 业务规则锁定

- 游客最多关注 5 个专题。
- 登录免费用户最多关注 10 个专题。
- 游客新建专题没有历史次数限制；新建后自动关注，所以仍受当前关注数上限限制。
- 登录后合并游客关注，但不能突破账号上限。
- 专题创建后不可修改、不可删除、不可归档。
- 用户唯一收口动作是取消关注。
- 新动态提醒只做红点和卡片高亮，不做通知中心、不显示未读数字。
