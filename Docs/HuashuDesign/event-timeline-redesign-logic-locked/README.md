# 事件时间轴 App 视觉重设计 · Logic Locked v3

本目录是 `huashu-design` 交付物，只做视觉设计，不修改 Flutter 业务代码。

## 文件

- `event-timeline-logic-locked-design.html`：高保真移动端设计板，可直接用浏览器打开。
- `brand-spec-logic-locked.md`：视觉规范、组件规则和业务边界。
- `screenshots/design-board.png`：完整设计板 PNG。
- `screens/*.png`：每个页面单独导出的 PNG。
- `figma-handoff.json`：Figma 重建参考 JSON，不是原生 Figma 源文件。

## 本版重点

- 推荐页分类文案严格保留：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 顶部入口、底部三 Tab、更多菜单四项、新建时间线流程保持当前截图逻辑。
- 不新增通知中心、我的专题、编辑、删除、归档等入口。
- 视觉采用浅蓝白、薄荷、淡紫、珊瑚的小面积状态色，避免沉重新闻和后台系统风格。
