事件时间轴 App 全新视觉设计

这份设计板以现有截图和冻结业务规则为结构依据，重做移动端视觉语言。目标是让“关注专题、查看时间线、发现推荐、新建专题”在轻快、明亮、可扫读的界面里成立，不改变接口、状态流和主流程。

A. Fresh Ribbon · 已深化 用奶油白、浅蓝、薄荷和珊瑚建立年轻感；卡片低半径，时间轴像轻量丝带。
B. Soft Ledger · 备用 更克制的信息表风格，适合后续做专业版，但会比当前目标更理性。
C. Pop Calendar · 备用 更强的日历感和色块节奏，适合活动型产品，但容易压过长文本。
