# 事件时间轴 App 全新视觉设计

本目录是 `huashu-design` 交付物，只做视觉与交互状态设计，不改 Flutter 业务实现。

## 文件

- `event-timeline-mobile-redesign.html`：高保真移动端设计板，可直接用浏览器打开。
- `brand-spec.md`：视觉规范、状态语义和业务边界。
- `screenshots/design-board.png`：本地渲染出的完整设计板截图。

## 覆盖页面

- 推荐页默认状态
- 推荐页下拉刷新状态
- 我的关注普通状态
- 我的关注有新动态红点和高亮状态
- 我的关注空状态
- 时间轴详情页
- 时间轴准备中状态
- 新建专题输入流程
- 新建专题定义确认 / 创建中流程
- 手机号验证登录流程
- 关注上限触发登录引导
- 网络失败轻提示状态
- 组件规范页
- 360 / 390 / 430px 移动端适配检查

## 业务约束

设计保留当前三 Tab 主结构，不新增通知中心、“我的专题”入口、编辑专题、删除专题或归档专题入口。
