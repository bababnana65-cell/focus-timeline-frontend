# 事件时间轴 App · Recommend Follow Left v29 视觉规范
> 日期：2026-04-29  
> 范围：基于 v28 的推荐卡片局部调整，不修改 Flutter 代码  
> 关键调整：去掉推荐卡片类型标签和左侧装饰图标，把已关注状态放到标题左侧

## 本版调整

v29 只调整推荐卡片状态表达：

- 推荐卡片不再显示左侧装饰图标。
- 推荐类型不再作为卡片内小标签出现，避免和顶部分类重复。
- 已关注状态改为标题左侧的紧凑标识。
- 未关注卡片不显示状态标识，保持文本优先。
- 保留左滑露出“关注 / 取消关注”的操作方式。
- 继承 v28：我的关注卡片继续突出最新节点消息。

## 组件规则

- `rec`：单列文本布局，去掉左侧 rank/icon 列。
- `rec-title-row`：标题行，承载已关注状态和标题。
- `followed-tag`：20px 高，浅绿色，位于标题左侧。
- `rec-tag`：不显示，推荐分类由顶部 Tab 表达。
- `rec-swipe-action`：左滑后显示关注或取消关注。
- 未读态：日期与摘要统一珊瑚色。
- 普通态：日期与摘要统一蓝色。
- 未读提示：只使用红点，不显示“新动态”文字。
- 进入详情：整张卡片点击，不依赖右侧箭头。

## 继承规则

- 我的关注卡片继承 v28：关注方向为辅助信息，最新节点消息为主阅读信息。
- 时间轴顶部统计区保持单行胶囊。
- 时间轴节点圆点保持 v23 放大样式。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
