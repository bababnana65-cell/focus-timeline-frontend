清透时间控制台

这版借鉴 Figma-inspired 的产品语言：近黑文字骨架、白色主界面、扁平 pastel 状态块、pill 控件和更克制的 8px 节奏。全程不使用渐变。

黑白骨架 主按钮、选中态和导航回到近黑系统，减少彩色竞争。
扁平色块 蓝、薄荷、珊瑚只做明确状态块，不做渐变和装饰背景。
工具质感 卡片圆角和阴影收紧，让页面更像可长期使用的产品界面。
