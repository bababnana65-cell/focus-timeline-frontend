# 事件时间轴 App · Flat Color v4 视觉规范
> 日期：2026-04-28  
> 范围：视觉设计交付，不修改 Flutter 代码  
> 关键要求：不使用渐变色；页面逻辑、入口、文案和业务规则保持当前截图语义

## 设计方向

方向名：**Flat Color Timeline / 纯色轻量时间轴**

- 用纯色浅蓝白做主背景，避免大面积暖黄、深蓝和科技渐变。
- 状态色只做小面积点缀：蓝色表示当前和主行动，薄荷表示准备中，淡紫表示排名和轻提示，珊瑚表示新动态。
- 卡片减少装饰，只保留轻边框、轻阴影和足够留白，让内容更容易扫读。
- 时间轴保持当前结构：左侧日期和数量圆点，右侧节点卡片。

## 不使用的视觉手法

- 不使用 `linear-gradient`。
- 不使用 `radial-gradient`。
- 不使用玻璃拟态大面积模糊。
- 不使用大面积深色背景。
- 不使用新闻客户端式厚重标题和压迫信息密度。

## 文案和逻辑锁定

- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 新建流程：`新建时间线 -> 关键词 -> AI 扩写 -> 专题定义确认`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。

## 组件规则

- 顶部按钮：44px 触控目标，纯白或纯蓝填充。
- 推荐分类：纯色选中态，不用发光或渐变。
- 专题卡片：普通卡片白底；新动态卡片浅蓝白底 + 红点 + 蓝色摘要高亮。
- 准备中状态：薄荷纯色进度条。
- 时间线：纯色浅灰线，节点数量用纯色圆点。
- Toast：深色纯色底，短反馈，不清空旧内容。
