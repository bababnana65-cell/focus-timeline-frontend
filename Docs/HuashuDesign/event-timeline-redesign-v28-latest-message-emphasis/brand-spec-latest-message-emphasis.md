# 事件时间轴 App · iOS Journal Latest Message Emphasis v28 视觉规范
> 日期：2026-04-29  
> 范围：基于 v27 的“我的关注”卡片局部调整，不修改 Flutter 代码  
> 关键调整：区分关注方向和最新节点消息的层级，并保证两者左对齐

## 本版调整

v28 只调整“我的关注”专题卡片文字层级：

- “关注方向”是辅助信息：更浅颜色、更低字重，帮助用户理解专题范围。
- “最新节点消息”是主要阅读对象：更高字重、浅色高亮，便于打开 App 后快速扫读。
- 最新节点消息的文字起点与关注方向文字起点严格对齐。
- 日期继续与最新节点消息同一行对齐。
- 继承 v27：去掉关注列表准备中字样、右箭头和重复“新动态”文字。

## 组件规则

- `topic-date`：11.5px，居中，不换行，通过渲染后测量与最新节点消息行对齐。
- `topic-body > .copy`：12.8px，较浅灰色，低字重，用于关注方向。
- `topic-latest-row`：12.9px，高字重，和日期同一阅读行，用于最新节点消息。
- `topic-latest-row .latest`：浅色高亮，左侧负外边距抵消内边距，让文字与关注方向左对齐。
- 未读态：日期与摘要统一珊瑚色。
- 普通态：日期与摘要统一蓝色。
- 未读提示：只使用红点，不显示“新动态”文字。
- 进入详情：整张卡片点击，不依赖右侧箭头。

## 继承规则

- 推荐卡片继承 v25：无常驻关注/查看按钮，左滑露出关注/取消关注。
- 时间轴顶部统计区保持单行胶囊。
- 时间轴节点圆点保持 v23 放大样式。
- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
