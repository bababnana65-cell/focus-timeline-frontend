# 事件时间轴 App 视觉重设计 · Gray Unselected Soft Coral v47

本目录是 `huashu-design` 交付物，只做视觉设计，不修改 Flutter 业务代码。

## 本版方向

- 单独新建 v47 目录，没有覆盖 v46、v45 或其他版本。
- 以 v46 为基础，只调整底部中间时间轴 Tab 的未选中灰色和选中指针配色。
- 对照 `Docs/HuashuDesign/app icon/timeline_app_icon_mark_transparent.svg` 恢复原始 6 层图标结构：阴影层、主色块、双层线条。
- 时间轴 Tab 选中态仍然没有外层圆形背景，直接显示彩色图标。
- 选中和未选中图标都统一为 `52px`，上沿对齐“我的关注”选中图标上沿，下沿对齐“我的关注”文字底部。
- 外层触控槽仍为 `58px × 52px`。
- 选中图标配色：绿色大面积色块 `#DFF2C7`，阴影层 `#86A866`，内部双层线条降饱和为柔和珊瑚棕 `#B87667 / #C98778`。
- 未选中图标同样保留原始阴影和双层线条结构，但整体改为和“推荐”Tab 一致的灰色体系 `#6F7484 / #8B93A1 / #F2F4F6`。
- 保留 v43：推荐页卡片里的“关注方向”说明文字也同步弱化。
- 推荐页 `.rec .copy` 使用和关注页方向文字同样的 `12.4px / 500 / #8C93A3`。
- “跟踪冲突升级、军事动作、外交回应、能源航运风险与谈判窗口变化”在推荐页和关注页保持同一视觉层级。
- “跟踪冲突升级、军事动作、外交回应、能源航运风险与谈判窗口变化”这类关注方向从 `12.8px / 580 / #6F7584` 降为 `12.4px / 500 / #8C93A3`。
- 最新节点消息保持高权重和浅色底，继续作为用户重点阅读内容。
- 保留 v41 的时间轴节点标题阅读列和右侧 `>` 控件关系。
- 时间轴节点标题和摘要改为稳定 `248px` 阅读列，并显式使用普通换行，避免 `text-wrap: pretty` 让不同节点第一行右边缘看起来不一致。
- 右侧 `>` 箭头按钮保持 `22px`，固定在日期胶囊同一行右侧，和文字阅读列分开。
- 保留 v38 的长文案换行检查。
- 专题标题、关注方向、最新摘要都换成自然换行的长文案，至少能看到两行排版。
- 推荐卡片和我的关注卡片都使用长文案，方便检查真实段落在手机卡片里的行高、间距和右边界。
- 时间轴节点标题和摘要宽度从 `264px` 收到 `248px`，为右侧箭头留出稳定空隙。
- 目标是让时间轴节点文字右间距接近“我的关注”卡片里的“跟踪冲突升级...”和“联调样本...”两行阅读文本。
- 保留 v36 的“我的关注”页右上角新建按钮颜色统一。
- 当前专题、普通列表、有新动态、新建流程背景中的“我的关注”右上角加号，统一为和空列表页一致的珊瑚色按钮。
- 使用 `Docs/HuashuDesign/app icon/timeline_app_icon_mark_transparent.svg` 的透明内标记，替换原 clock 图标。
- 选中态跟随时间轴页绿色主题，未选中态使用柔和灰色，避免深色块。
- 移除底部中间 Tab 的“时间轴”文字，只保留 App 图标。
- 图标放大为独立入口，上下视觉高度对齐“我的关注”的图标 + 文字组合。
- 时间轴节点箭头继续保留 v33 的卡片内原位置。
- 保留 v32 的主要阅读文本右边界对齐规则。
- 保留 v31 的“正在查看”标签移除方案。
- 保留 v30 的推荐页调整：卡片内类型标签隐藏，“已关注”在左侧图标下方。
- 保留未读红点提示。

- 不使用渐变色。

- 保留当前截图里的入口、页面结构、分类文案和业务规则。
