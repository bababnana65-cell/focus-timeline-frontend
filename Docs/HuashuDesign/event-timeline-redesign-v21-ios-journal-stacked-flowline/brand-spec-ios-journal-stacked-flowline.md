# 事件时间轴 App · iOS Journal Stacked Flowline v21 视觉规范
> 日期：2026-04-28  
> 范围：基于 v20 的时间轴局部重设计，不修改 Flutter 代码  
> 关键调整：日期与节点流线改为上下关系，避免左侧横向局促

## 本版调整

v21 重新处理时间轴左侧信息：

- 日期不再放在左侧窄列内。
- 日期作为节点卡片顶部的小标签出现，保持在节点内容之前。
- 左侧只保留连续流线和节点数量圆点。
- 流线宽度保持轻量，不争夺正文空间。
- 右侧节点卡片获得更完整的阅读宽度。
- 节点标题、摘要和右侧箭头结构不变。

## 时间轴组件规则

- 日期标签：统一天蓝色，使用浅主题底色，位于节点标题上方。
- 流线：统一青柠绿，左侧纵向贯穿节点组，不使用渐变。
- 节点数量圆点：挂在流线上；普通节点使用时间轴主题色，重大节点使用珊瑚色。
- 节点摘要：统一 `--muted`，不因节点重要程度变色。
- 事件卡片：保留 v17 的卡片风格和轻阴影。

## 继承规则

- 推荐分类：`可能关心 / 当前热门 / 随机看看 / 历史记录`。
- 底部 Tab：`我的关注 / 时间轴 / 推荐`。
- 时间轴更多菜单：`新建时间线 / 分享 / 取消关注 / 时间顺序`。
- 新建流程：`新建时间线 -> 关键词 -> AI 扩写 -> 专题定义确认`。
- 不新增通知中心、我的专题、编辑、删除或归档入口。
