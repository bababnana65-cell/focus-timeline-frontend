# 事件时间轴 App 视觉重设计 · iOS Journal Stacked Flowline v21

本目录是 `huashu-design` 交付物，只做视觉设计，不修改 Flutter 业务代码。

## 本版方向

- 单独新建 v21 目录，没有覆盖 v20 或其他版本。
- 以 v20 为基础，重新设计时间轴左侧节点流线和日期关系。
- 日期从左侧窄列移到节点卡片顶部，形成“上方日期标签 + 下方节点内容”。
- 左侧只保留细流线和数量圆点，减少局促感。
- 节点卡片获得更完整的正文宽度，标题和摘要阅读更顺。
- 保留 v17/v20 的整体色彩、语义色规则和页面视觉基调。
- 不使用渐变色。
- 不使用不规则多边形色块。
- 不使用黑色或深色填充块；深色只用于正文文字和必要线性图标。
- 保留当前截图里的入口、页面结构、按钮文案、分类文案和业务规则。

## 文件

- `event-timeline-ios-journal-stacked-flowline.html`：高保真移动端设计板。
- `brand-spec-ios-journal-stacked-flowline.md`：上下日期流线视觉规范。
- `screenshots/design-board.png`：完整设计板 PNG。
- `screens/*.png`：单页高保真 PNG。
- `figma-handoff.json`：Figma 重建参考 JSON，不是原生 `.fig` 文件。

- 我的关注：普通列表 / 有新动态 / 当前专题 / 空列表
